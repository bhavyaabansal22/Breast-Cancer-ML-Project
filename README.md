# Breast Cancer Classification and Clustering

## 📌 Objective
To classify tumors as benign or malignant using machine learning models and to analyze patterns in the dataset using clustering techniques.

## 📊 Dataset
The Breast Cancer Wisconsin Dataset is used, which contains 30 numerical features representing characteristics of cell nuclei and a target variable indicating whether the tumor is benign or malignant.

## ⚙️ Models Used

### Classification Models
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

### Clustering Models
- K-Means Clustering
- Hierarchical Clustering
- DBSCAN

## 🔍 Methodology
- Data preprocessing and feature scaling using StandardScaler
- Splitting dataset into training and testing sets
- Training multiple classification models
- Evaluating using accuracy, precision, recall, and confusion matrix
- Applying clustering algorithms
- Evaluating clustering using silhouette score

## 📈 Results
- Logistic Regression achieved the highest classification accuracy
- DBSCAN showed the best clustering performance based on silhouette score
- Visualizations were used for better understanding of model performance
